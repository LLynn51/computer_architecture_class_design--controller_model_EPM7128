# 题目A：基于Altera CPM7128的模型机系统控制器设计

## 基础层次—硬布线控制器设计

按照给定指令格式、数据格式与数据通路，以Verilog/VHDL设计实现基于硬布线的顺序模型处理器；支持TEC-8基本指令集（除中断之外）；在TEC-8实验平台完成组装、调试并运行测试程序。

## 进阶层次—流水型硬布线控制器设计

将顺序控制器改造为流水线结构，分析并解决数据冒险与控制冒险问题；对时钟频率与流水线吞吐量进行量化分析，比较流水前后的性能差异。

- 实现难点：硬件平台上的冒险处理，需观察、分析并解决吞指令现象。
 
## 拓展层次—带中断功能的硬布线控制器设计

在必选题基础上设计实现支持中断响应与返回的硬布线控制器。

- 设计难点：EPM7128芯片中未接入LIAR和IABUS信号，无法以IAR寄存器实现中断现场保存及返回，需自行设计替代方案。
